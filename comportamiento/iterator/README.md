# Iterator

**Categoría:** De comportamiento 🟡 · **Responsable:** _Juan Manuel Giraldo_

## 📌 Problema
_Recorrer colecciones complejas (árboles/grafos) requiere algoritmos difíciles. Si los metes en la colección, la saturas; si los dejas en el cliente, lo acoplas a la estructura._

## 💡 Solución
_Separar el recorrido en un objeto independiente (Iterador). Este guarda el progreso y da una interfaz única para extraer los datos paso a paso._

## 🧪 Ejemplo de uso
_Es un programa en Java que usa un asistente (Iterador) para recorrer y mostrar los libros de un Librero uno por uno. Su objetivo es proteger la lista interna de libros, evitando que el código exterior la vea o la modifique directamente_
